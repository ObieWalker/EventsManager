## TECHNOLOGIES USED

  * Front-end: React/Redux + Bootstrap (Yet to be Implemented)
  * Back-end: Node/Expressjs + Sequelize/Postgres
  * Libraries: nodemon, Babel, eslint, etc.
  * Test: Mocha/Chai
